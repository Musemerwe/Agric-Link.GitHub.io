This is the Agric Link

Agric Link offers a simple and efficient platform for farmers to list their products, set prices, and reach a wider audience. Buyers can browse listings, compare prices, and purchase directly from the source, ensuring fresh and high-quality products. Join us in fostering a sustainable food system that benefits both farmers and consumers.

Technical Details

  1. Firebase Authentication: Manages user registration and login, ensuring secure access to the app.
  2. Firestore Database: Stores user information, product details, and chat messages.
  3. Firebase Storage: Stores images uploaded by farmers and provides URLs to be used in the app.
  4. Image Picker: Allows farmers to select images from their device to upload.
  5. Provider: Manages state within the app to ensure data consistency and smooth user experience.
  6. Google Maps Integration: The product_map_page.dart uses Google Maps API to display the location of the farmer.

By following these workflows, the app provides a seamless experience for both farmers and suppliers, facilitating efficient communication and transactions in the agricultural sector.
